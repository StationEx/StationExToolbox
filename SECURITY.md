# StationEx Security Policy

This policy applies to all StationEx repositories.


## Reporting

To report a security issue, please use the GitHub Security Advisory ["Report a Vulnerability"](https://github.com/stationex/stationex.toolbox/security/advisories/new) tab on the [StationEx.Toolbox Security Advisories](https://github.com/stationex/stationex.toolbox/security/advisories) page.

You should receive a response to your report within 48 hours.


## Escalation

If you do not receive a response to your report within 48 hours, please email security@stationex.dev to escalate the matter.


## Remediation

If you have a patch for a security issue that is unreported, report the issue and mention you have a patch in the report.

If you have a patch for a security issue that is reported, submit a pull request and reference the vulnerability that the patch remediates.
